The `data-ui-options="contentOptions.imageCarousel"` is providing a global JavaScript variable that eventually holds options to apply to the component. In this case the variable is defined in the [./app/main.js](./app/main.js) as follow: ```javascript var contentOptions = { // ... imageCarousel: { on: { 'page:tap': function(e, pageIndex, tp) { // open the full screen image browser // ... } } } // ... } ``` so, it basically indicates what to do when an image is clicked and in this case it will open it full screen with the *Image Browser* component, which is another component that is loaded in the `index.html` file as follow: ```html
